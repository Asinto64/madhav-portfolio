# Portfolio Site

Starter portfolio built with:

- Next.js
- React
- TypeScript
- Tailwind CSS
- Motion for React

## Run

Install dependencies:

```bash
npm install
```

Start dev server:

```bash
npm run dev
```

## Suggested first edits

1. Replace the hero headline with your name and role.
2. Add 3 real projects.
3. Add your contact links.
4. Tune colors and motion to match your style.
