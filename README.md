# Madhav Gupta | Premium Portfolio

This repository contains the source code for Madhav Gupta's cinematic, high-end professional portfolio website.

## Project Approach & Design Philosophy

The main objective of this project is to provide a $2000-grade cinematic web experience that showcases Madhav's work with maximum impact. The design emphasizes premium visual excellence and modern UI/UX trends suitable for direct domain deployment.

### Key Features & UX Strategies
- **Cinematic Dark Mode Aesthetic:** Deep, immersive dark themes with subtle gradients and glowing accents that provide a high-end feel.
- **Dynamic Scroll Animations:** Utilizes sophisticated scroll-linked micro-animations, fade-ins, and parallax effects to keep the user engaged.
- **3D Graphics & Glassmorphism:** Incorporates custom 3D asset integrations and frosted-glass components for depth and modern appeal.
- **Modern Typography:** Carefully selected, clean font pairings that ensure readability while exuding professionalism.
- **Interactive "Bento" Grids:** A structured, easy-to-digest layout format for showcasing projects, skills, and interests interactively.

## Technology Stack

- **Framework:** Next.js & React
- **Language:** TypeScript
- **Styling:** Tailwind CSS (with highly customized utility patterns)
- **Animation:** Motion for React (Framer Motion)

## Local Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
